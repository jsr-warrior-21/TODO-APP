# TaskFlow Pro 🚀

A modern, production-ready task management application built with React and FastAPI.

## ✨ Features

- ✅ **Authentication** - Login/Register with JWT
- 🌙 **Dark Mode** - Toggle between light and dark themes
- 📝 **Task Management** - Create, Read, Update, Delete tasks
- ⭐ **Important Tasks** - Mark tasks as important
- ✅ **Task Completion** - Mark tasks as complete/incomplete
- 🔍 **Search & Filter** - Search tasks with real-time filtering
- 📊 **Sorting** - Sort by date, priority, alphabetically
- 📦 **Bulk Actions** - Select multiple tasks and perform actions
- 📜 **Activity Log** - Track all task activities
- 🔥 **Task Streak** - Track daily completion streak
- 🎉 **Confetti Celebration** - Celebrate task completion
- 📅 **Due Date Badges** - Color-coded due date indicators
- 📤 **Export CSV** - Export tasks to CSV
- ⌨️ **Keyboard Shortcuts** - Ctrl+N, Ctrl+F, ESC

## 🛠️ Tech Stack

### Frontend
- React 18
- Vite
- Tailwind CSS
- Framer Motion (Animations)
- React Hot Toast (Notifications)
- Lucide React (Icons)

### Backend
- FastAPI
- SQLAlchemy
- JWT Authentication
- PostgreSQL (or SQLite)

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- Python 3.9+
- PostgreSQL (optional, SQLite works)

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
